Analisis Sentimen dari File CSV

File harus berformat CSV dengan kolom 'text' yang berisi teks untuk dianalisis.
Petunjuk Penggunaan
  1. Upload file CSV yang berisi data teks yang ingin dianalisis
  2. Pastikan file CSV memiliki kolom dengan nama 'text'
  3. Sistem akan menganalisis sentimen setiap teks
  4. Hasil analisis dapat diunduh dalam format CSV

Contoh struktur CSV:

text
"Produk ini sangat bagus"
"Pelayanannya kurang memuaskan"
"Saya netral saja"
